# URL Routing

## [Master]

### GET Admin

/master/user?role=admin&token={access_token}

### GET Kurir

/master/user?role=kurir&token={access_token}

### GET User

/master/user?role=pengguna&token={access_token}

### GET Order

/master/order?token={access_token}


### GET Tracking

/master/tracking?kurir={id}
